# Nmap Scan Lab

## Objective
Identify open ports and services on a target machine.

## Command Used
nmap -sV 192.168.56.101

## Results
- Port 21: FTP
- Port 22: SSH
- Port 80: HTTP

## Analysis
Multiple services exposed, potential vulnerabilities in FTP and outdated web server.

## Conclusion
Target machine has multiple entry points that could be exploited.
